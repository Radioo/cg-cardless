# CG Cardless

Helper app for cardless login in supported games. Set your card ID and just scan the code.

As a bonus, android build also has built-in FeliCa host card emulation you can use.

Built with [Expo](https://expo.dev) and React Native.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm start
   ```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Start the Expo dev server |
| `npm run android` | Run on Android |
| `npm run ios` | Run on iOS |
| `npm run web` | Run in browser |
| `npm test` | Run tests |
| `npm run test:watch` | Run tests in watch mode |
| `npm run test:ci` | Run tests for CI |
| `npm run lint` | Lint the codebase |
